1、svn - 服务器创建一个svn项目仓库 -- 客户端检出项目仓库 -- 添加 -- 提交 -- 更新
2、svn - 集中式版本管理工具，一个服务器端对应多个客户端，安全性不高，svn工作不能断网

3、git：分布式 版本管理工具，一台电脑既可以是服务器端也可以是客户端，工作过程中可以断网，提交的时候还是必须要有网络的

使用git的步骤：安装
	1、Git-2.18.0-64-bit.exe
	2、TortoiseGit-2.6.0.0-64bit.msi 按不安装都无所谓，因为就是小乌龟
		给我们提供一些简化操作，可以显示出各种图标

对着你的项目文件右键，打开git bash here，git的命令行
1、初始化项目文件 git init
？代表 文件没有添加到本地的git上 文件没有被git所管理

2、查看某些文件或文件夹是否被git所管理
   git status

3、将某个文件添加到本地的git上 被git管理
   git add 文件名
+ 表示文件已经被提交到本地的git上，还没有提交到git服务器上，说明已经被git管理

4、批量添加某些文件被git管理
   git add .

5、忽略某些文件不被git管理
   创建一个 .gitignore 文件 
   通过 touch .gitignore
   将忽略的文件 全称 写入到此文件之中
